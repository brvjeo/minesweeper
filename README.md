# Minesweeper

## Run
_Choose one of these 3 ways:_
- 1
```
$ npm i
$ npm run start
```
- 2

Open `index.html` file in root folder.

- 3

Also available on [**GitHub Pages**](https://brvjeo.github.io/minesweeper/).

## Stack

- **React**
- **Typescript**

## Project structure

`utils/index.ts` - application logic, helpers

`components/Game/Game.tsx` - main component, container for event handlers

```
src/
├── components
│   ├── App.tsx
│   ├── Counter
│   │   └── Counter.tsx
│   ├── Game
│   │   └── Game.tsx
│   ├── Grid
│   │   └── Grid.tsx
│   ├── GridCell
│   │   └── GridCell.tsx
│   └── Timer
│       └── Timer.tsx
├── constants
│   └── index.ts
├── hooks
│   └── useTimer.ts
├── index.tsx
├── types
│   └── index.ts
└── utils
    └── index.ts
```
