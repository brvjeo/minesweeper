# Minesweeper

## Run

```
$ npm i
$ npm run start
```

Also available on **GitHub Pages**.

## Stack

- **React**
- **Typescript**

## Project structure

`utils/index.ts` - application logic, helpers

`components/Game/Game.tsx` - main component, container for event handlers

```
src/
├── components
│   ├── App.tsx
│   ├── Counter
│   │   └── Counter.tsx
│   ├── Game
│   │   └── Game.tsx
│   ├── Grid
│   │   └── Grid.tsx
│   ├── GridCell
│   │   └── GridCell.tsx
│   └── Timer
│       └── Timer.tsx
├── constants
│   └── index.ts
├── hooks
│   └── useTimer.ts
├── index.tsx
├── types
│   └── index.ts
└── utils
    └── index.ts
```
